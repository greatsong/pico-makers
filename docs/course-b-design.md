나만의 대시보드 측정기를 만드세요.

## 이런 걸 만들 수 있어요
┌─────────────────┬──────────────────┬─────────────────┐
│ 교실 온도        │ 소음 레벨         │ 자동 조명        │

## 2. 피코 메이커스 보완 매핑

교재의 각 섹션에서 피코 메이커스가 어떻게 보완하는지:

### 교재: "센서를 골라 연결하세요"
→ **피코 메이커스**: 배선 탭
- Grove Shield 포트맵 (어디에 꽂을지 시각적 안내)
- 준비물 체크리스트 (체크하며 확인)
- 배선 체크리스트 (연결 순서대로 체크)
- 핀 충돌 감지 (센서 2개+ 시 자동 경고)

### 교재: "코드를 작성하세요"
→ **피코 메이커스**: 코드 탭
- 센서별 예제 코드 (줄별 설명 ON/OFF)
- 통합 코드 자동 병합 (센서 2개+ 시)
- 복사 버튼 → Thonny에 붙여넣기

### 교재: "바이브 코딩을 활용하세요" (Lv.2+)
→ **피코 메이커스**: AI 채팅
- AI가 현재 센서 조합을 이미 알고 있음
- 프롬프트 없이도 "OLED 대시보드 만들어줘"만으로 맥락 인식
- 에러 메시지 붙여넣기 → 자동 진단

### 교재: "데이터를 분석하세요" (Lv.3)
→ **피코 메이커스**: AI 채팅
- "이 CSV 데이터를 분석해줘" → matplotlib 코드 생성
- 가설 검증 논리 코칭

### 교재: 평가 기준
→ **피코 메이커스**: 팁 탭
- 필수/우수/탁월 기준 실시간 확인
- 타임라인으로 "지금 뭘 해야 하는지" 안내

```
  → DHT20 + OLED 자동 추가
  → 배선 탭: Grove Shield I2C1 포트 안내 + 준비물 체크
  ↓
[교재] Step 1: 센서 읽기 코드 참고
[피코 메이커스] 코드 탭: DHT20 예제 코드 복사 → Thonny에 붙여넣기
  ↓
[피코 메이커스] AI 채팅: "OLED에 온도를 막대그래프로 보여주는 코드 만들어줘"
  → AI가 DHT20+OLED 맥락을 알고 코드 생성
  ↓
[교재] Step 4: 조건 판단 코드 참고
[피코 메이커스] AI: "if문으로 온도 상태를 나누려면?"
  ↓
[교재] Step 5: CSV 저장 코드 참고
[피코 메이커스] AI: "CSV 저장 기능 추가해줘"
  ↓
[교재] 산출물 체크리스트로 최종 확인
[피코 메이커스] 팁 탭: 평가 기준 확인
```

| 사진 분석 기능 활용 | 낮음 | OLED 사진 찍어서 AI에게 "이거 맞아?" |


## 6. 요약: 교재와 피코 메이커스의 역할 분담

```
교재 = 지도 (어디로 가야 하는지 알려줌)
피코 메이커스 = 내비게이션 (실시간으로 경로를 안내함)

교재가 없으면: 피코 메이커스는 도구일 뿐, 방향이 없음
피코 메이커스가 없으면: 교재대로 할 수 있지만 헤매는 시간이 길어짐
둘이 함께면: 효율적이고 풍부한 학습 경험
```
